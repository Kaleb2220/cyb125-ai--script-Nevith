# CYB125 AI Script
## This project is a script created for CYB125
This scripts purpose is to tell if a password is weak or strong 
This project was written in PowerShell
## Tools Used 
Ai tool used: Copilot
Editor used: Notepad, VScode
## How it works
This script asks you to enter a password, it asks you to confirm a password, then it evaluates the confirmed passwords by checking the for length, uppercase letters, lowercase letters, numbers, and symbols
Then it calculates a total score, the score setermines whether the passwors is weak (0-3), meduim (4-6), or strong (7+).
It prints the results in color. Red = Weak, Yellow = Meduim, Green = Strong.
And it lists the reasons like: -Contains uppercase letters, contains symbols, good length (12+)
It is a mini password quality checker that you can run locally
